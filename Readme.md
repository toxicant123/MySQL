## SQL的分类

DDL:数据定义语言：CREATE \ ALTER \ DROP \ RENAME \ TRUNCATE


DML:数据操作语言：INSERT \ DELETE \ UPDATE \ SELECT （重中之重）


DCL:数据控制语言：COMMIT \ ROLLBACK \ SAVEPOINT \ GRANT \ REVOKE
   
## SQL语句执行顺序

|        语句组        | 执行次序 |
|:-----------------:|:----:|
|    select 查询列表    |  7   |
|    from 表1 别名     |  1   |
|   连接类型 join 表2    |  2   |
|      on 连接条件      |  3   |
|     where 筛选      |  4   |
|   group by 分组列表   |  5   |
|     having 筛选     |  6   |
|   order by排序列表    |  8   |
| limit 起始条目索引，条目数; |  9   |
